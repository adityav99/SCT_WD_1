Responsive Landing Page
======================

Files included:
- index.html
- style.css
- script.js

How to use:
1. Unzip the archive.
2. Open index.html in your browser.
3. For local development with live reload, serve the folder with a development server (e.g., `npx http-server`).

Features:
- Fixed top navigation that changes style on scroll.
- Hover effects on menu items.
- Mobile hamburger menu for small screens.
- Smooth scrolling.
